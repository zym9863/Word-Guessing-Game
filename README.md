# 猜单词小游戏

这是一个简单的猜单词小游戏，使用 HTML 和 JavaScript 实现。

## 项目结构

- `index.html`：项目主页面，包含游戏界面。
- `game.js`：游戏的主要逻辑脚本。

## 如何运行

1. 下载本项目所有文件。
2. 用浏览器打开 `index.html` 文件即可开始游戏。

## 功能简介

- 随机生成单词，玩家输入猜测。
- 判断玩家输入是否正确，并给出提示。
- 支持多轮游戏。

## 适用环境

- 现代浏览器（如 Chrome、Edge、Firefox 等）
